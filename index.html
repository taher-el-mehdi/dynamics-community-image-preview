üíæ Installation Guide

Quick and easy setup in developer mode

Chrome / Edge Installation

  1. Download the extension from GitHub or clone the repository
  2. Open your browser and navigate to chrome://extensions/ or edge://extensions/
  3. Enable "Developer mode" using the toggle in the top right corner
  4. Click "Load unpacked" button
  5. Select the community-image-viewer folder
  6. Visit community.dynamics.com/forums and start previewing images!

üìù Note: You'll need to create icon files (16x16, 48x48, 128x128) or temporarily remove icon references from manifest.json. See the ICONS_README.txt file for details.
